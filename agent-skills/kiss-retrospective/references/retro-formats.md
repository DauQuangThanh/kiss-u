# Retrospective formats — quick reference

| Format | Columns | When to use |
|---|---|---|
| **WWD** (What went Well / What Didn't / What to Do next) | 3 | Default; safe for most sprints |
| **Wwwdt** (Well / Wonder / Different / Try) | 4 | When the team wants space to ask questions ("wonder") |
| **Mad / Sad / Glad** | 3 | When morale is the key signal |
| **4 Ls** (Liked / Learned / Lacked / Longed-for) | 4 | Good for learning-oriented sprints |
| **SAFE** (Safe / Appreciative / Fix / Experiment) | 4 | Use after conflict or difficult decisions |
| **Start / Stop / Continue** | 3 | Simple, action-focused |

## Picking a format

- If the team is new to retros → WWD.
- If the same issues keep surfacing → switch format to break
  patterns; Mad/Sad/Glad or SAFE often help.
- If morale is low → Appreciative-leaning formats.

## AI-authoring note

The skill preserves the team's wording — don't paraphrase into
manager-speak. Group near-duplicates, but leave the original voice
visible in each grouped item.
